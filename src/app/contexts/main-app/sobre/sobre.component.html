Apoie o projeto

Estamos em busca de hospedagem para este serviço web. Ao hospedar o projeto, você permitirá que empresas e instituições de ensino da sua região tenham acesso a uma plataforma que pode colaborar na melhoria do processo de divulgação de vagas de estágio para estudantes.

O projeto é desenvolvido com a filosofia de código aberto, o que significa que você tem liberdade para utilizar, contribuir e aprimorar o serviço. Também estamos disponíveis para fornecer a assistência necessária para hospedagem e integração inicial.

Desenvolvimento

Este projeto foi desenvolvido na disciplina Projeto Integrador - 2024.1, como requisito para conclusão do Curso Técnico em Desenvolvimento de Sistemas - Instituto Federal de Santa Catarina - Câmpus Florianópolis
